The Forward Problem: Finite Differences

−u″(x) = f(x)  →  Au = f  →  solve the linear system
Matrix A (tridiagonal)

Discretise on a uniform grid with spacing h = 1/(N+1). At each interior point, the second derivative becomes:

−u″(xᵢ) ≈ (−u[i−1] + 2u[i] − u[i+1]) / h²

This gives the sparse linear system Au = f, where A is the tridiagonal matrix shown left. The boundary conditions u(0) = u(1) = 0 are built in automatically.

For each new forcing f, we solve Au = f using the Thomas algorithm — an O(N) tridiagonal solver.
